# beecrowd Exercises

Solutions for programming challenges from [beecrowd](https://www.beecrowd.com.br/), organized by category, language, and database engine.

This repository is part of my continuous practice in algorithms, problem solving, and SQL. The goal is to keep the solutions simple, readable, and easy to revisit.

## Technologies

| Area | Technologies |
| --- | --- |
| Programming exercises | JavaScript, TypeScript |
| SQL exercises | MySQL, PostgreSQL |
| Runtime and tooling | Node.js, TypeScript |

## Contents

- **Beginner problems**: JavaScript and TypeScript solutions for introductory programming exercises.
- **SQL problems**: MySQL and PostgreSQL solutions for database challenges.
- **Incremental practice**: New exercises can be added over time while keeping the same folder structure.

## Repository Structure

```text
.
├── beginner/
│   ├── javascript/
│   └── typescript/
└── sql/
    ├── mysql/
    └── postgresql/
```

## Running Solutions

JavaScript solutions can be executed with Node.js:

```bash
node beginner/javascript/1000.js
```

TypeScript solutions can be checked or compiled using the local TypeScript setup:

```bash
npm run typecheck
```

SQL solutions are grouped by database engine and can be run in a compatible MySQL or PostgreSQL environment.
